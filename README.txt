🐳 Zabbix 7.0.0 Alpha 9 – Docker Compose

Este repositório contém uma estrutura personalizada para executar o Zabbix 7.0.0 Alpha 9 utilizando Docker Compose, com foco em modularidade, controle de build e facilidade de manutenção.
🔧 Recursos incluídos:

    Zabbix Server com build próprio via pacote .deb

    Containers organizados seguindo a estrutura oficial do repositório do Zabbix Docker

    Suporte para PostgreSQL, frontend, agent, grafana e proxy

    Arquivos .env e entrypoint.sh adaptáveis por ambiente

    Ideal para testes, validações de features novas e ambientes de homologação

🚨 Atenção:

    Esta é uma versão Alpha do Zabbix 7.0.0, indicada somente para fins de teste.
    Não é recomendada para ambientes de produção.

📁 Estrutura baseada em:

🔗 zabbix/zabbix-docker - versão 7.2
💬 Contribuições e sugestões são bem-vindas!
